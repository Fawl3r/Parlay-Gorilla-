# Preview Images Folder

This folder contains images that will automatically rotate as link preview images when sharing ParlayGorilla.com links on social media.

## How It Works

- **Automatic Discovery**: Any image file you place in this folder will automatically be included in the rotation
- **Daily Rotation**: Images rotate once per day based on the date
- **Supported Formats**: PNG, JPG, JPEG, WEBP, GIF, SVG
- **Alphabetical Order**: Images are sorted alphabetically for consistent rotation order

## Adding Images

Simply drop image files into this folder. They will be automatically picked up and included in the rotation.

## Image Recommendations

- **Size**: 1200x630 pixels (optimal for social media previews)
- **Format**: PNG or JPG recommended
- **File Size**: Keep under 1MB for faster loading
- **Naming**: Use descriptive names (they're sorted alphabetically)
